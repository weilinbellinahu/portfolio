⚠️ Ce qu'il faut retenir

  • Depuis 2021, GitHub n'accepte plus les mots de passe — utiliser un Token ou une clé SSH
  • Ne jamais partager un Personal Access Token — même dans un chat IA. GitHub le révoque automatiquement
  • Le connecteur GitHub MCP nécessite une nouvelle conversation pour s'activer
  • Après la configuration DNS, patienter (10 min à quelques heures) — l'avertissement SSL pendant ce délai est normal
  • Une fois le certificat SSL généré, cocher « Enforce HTTPS » dans les paramètres GitHub Pages
