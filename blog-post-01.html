Ce qu'il faut retenir

  • GitHub n'accepte plus les mots de passe depuis 2021 — utiliser SSH
  • Ne jamais partager un Token dans un chat — GitHub le révoque automatiquement
  • Le connecteur GitHub MCP nécessite une nouvelle conversation
  • Après DNS, patienter 10 min à quelques heures — SSL warning est normal
  • Cocher Enforce HTTPS dans GitHub Pages après génération du certificat
