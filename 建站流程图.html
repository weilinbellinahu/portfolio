⚠️ 本次遇到的坑 — À RETENIR

  • GitHub 从 2021 年起不支持密码推送，必须用 Token 或 SSH Key
  • Personal Access Token 绝对不能发给任何人——包括 AI 对话，GitHub 会自动撤销
  • GitHub MCP 连接器需要新对话才能激活，不能在同一对话中连接后立即使用
  • 自定义域名配置后需等待 DNS 传播（10分钟～数小时），期间出现 SSL 警告属正常
  • DNS 生效 + SSL 证书生成后，记得回 GitHub Pages 勾选 Enforce HTTPS
